# My Personal Website
Using a [StartBootstrap](https://startbootstrap.com/template-overviews/creative/) theme and [Gulp](http://gulpjs.com/) for the build system

## Using Gulp
```
$ npm install gulp -g
$ npm init
$ npm install gulp --save-dev
```
##To Run Gulp
```
$ gulp
```
